## jquery评论晒照插件

## 用法
1、导入插件

    <link rel="stylesheet" type="text/css" href="src/ss-imgbox.css">
    <script type="text/javascript" src="src/js/jquery-1.7.1.min.js"></script>
    <script type="text/javascript" src="src/js/jquery.ss-imgbox.js"></script>

2、调用

    $(function(){
        $("#ss-imgbox").ssImgBox();
    });
